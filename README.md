
# Streaming Image Sequence
Streaming Image Sequence is a plug-in for Unity Timeline to use sequential pictures without making Unity 2D Sprite.
## Target Environment
- Unity version: 2019.2+
- Platform: 
 - Mac Standalone
 - Mac Editor
 - Windows Standalone
 - Windows Editor
